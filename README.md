To run locally:

1. Install deps: `pip install -r requirements.txt`
2. Launch the desktop app: `python app.py`
3. A PyWebView window opens with the HTML/CSS/JS UI (pure Python backend; no C++ build needed).
4. Watch the console logs for progress entries (im_vs_z, roots_vs_beta, phase, eigen) to confirm button actions.
